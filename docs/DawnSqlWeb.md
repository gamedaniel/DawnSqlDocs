# DBeaverWeb
## 1、启动程序
从命令行进入 DBeaverWeb-1.0-SNAPSHOT.war 所在的目录。输入命令启动 DBeaverWeb 程序
```shell
> java -jar DBeaverWeb-1.0-SNAPSHOT.war
```
## 2、登录页面
在浏览器中输入 http://localhost:8086/login
